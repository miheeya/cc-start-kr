# 보너스: 실전 업무 따라하기

> 레슨 1~12에서 배운 기능을 실제 업무에 조합해서 사용해 봅시다.

## 시나리오 1: 회의록 요약 → 이메일 작성

**상황:** 회의가 끝났습니다. 회의록을 요약하고, 부재자에게 공유 이메일을 작성해야 합니다.

**따라하기:**

```
1단계: 파일 참조로 회의록 요약 (레슨 3)
> @samples/회의록_예시.txt 핵심 내용만 3줄로 요약해줘

2단계: 이메일 작성 요청 (레슨 1)
> 위 요약을 바탕으로, 부재자에게 보낼 공유 이메일을 작성해줘.
  존댓말로, 핵심 결정사항과 다음 액션을 포함해줘.

3단계: 파일로 저장
> 이 이메일을 '회의공유_0403.txt' 파일로 저장해줘
```

## 시나리오 2: 보고서 분석 → 인사이트 도출

**상황:** 1분기 매출 보고서를 받았습니다. 핵심 인사이트를 뽑아 상사에게 보고해야 합니다.

**따라하기:**

```
1단계: 보고서 분석 (레슨 3)
> @samples/보고서_예시.txt 이 보고서에서 가장 주목할 만한 점 3가지를 알려줘

2단계: 심층 분석 (레슨 6 — effort 활용)
> /effort high
> 이 데이터에서 2분기에 주의해야 할 리스크와 기회를 분석해줘

3단계: 보고용 요약
> 위 분석을 1페이지 분량의 경영진 보고 형식으로 작성해줘
```

## 시나리오 3: 이메일 검토 → 수정 → 번역

**상황:** 파트너사에 보낼 이메일 초안을 검토하고, 영어로도 번역해야 합니다.

**따라하기:**

```
1단계: 이메일 검토 (레슨 3)
> @samples/이메일_초안.txt 이 이메일에서 개선할 점을 알려줘

2단계: 수정 요청
> 피드백을 반영해서 이메일을 수정해줘

3단계: 영어 번역
> 수정된 이메일을 비즈니스 영어로 번역해줘

4단계: 저장 (레슨 1)
> 한국어 버전과 영어 버전을 각각 파일로 저장해줘
```
