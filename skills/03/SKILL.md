# 내 파일들과 대화하기

> 되돌리기를 배웠으니, 이제 안심하고 파일을 다뤄봅시다.

## 이게 뭔가요?

파일 내용을 복사해서 붙여넣을 필요 없이, `@`를 입력하면 파일을 검색해서 바로 첨부할 수 있습니다.
스마트폰에서 `@친구이름`으로 태그하는 것과 비슷합니다.

## 사용 예시

```
> @회의록 요약해줘            ← @ 입력하면 파일명 검색 시작
  ❯ 회의록_0405.pdf
    회의록_0401.pdf

> @회의록_0405.pdf 요약해줘
  ◐ Reading 회의록_0405.pdf...
  이 회의의 주요 안건은 3가지입니다:
  1. 신규 프로젝트 일정 확정
  2. 예산 검토
  3. 다음 회의 날짜 조정
```

## 이렇게도 쓸 수 있어요

- **`@보고서.xlsx`** — 엑셀 데이터를 분석해달라고 요청
- **`@계약서.pdf`** — 계약서 내용을 확인하거나 요약
- **`@폴더이름/`** — 폴더 전체를 한 번에 첨부
- 파일 이름의 일부만 입력해도 자동으로 찾아줍니다

## 직접 해보세요!

연습용 파일이 `samples/` 폴더에 준비되어 있습니다:

1. `@samples/회의록_예시.txt 요약해줘` 라고 입력해 보세요
2. `@samples/보고서_예시.txt 핵심 수치만 알려줘` 라고 물어보세요
3. `@samples/이메일_초안.txt 개선할 점을 알려줘` 라고 요청해 보세요

## 팁

- 한국어로 자연스럽게 말하면 됩니다. "이 파일 요약해줘" 처럼요.
- 여러 파일을 동시에 참조할 수도 있습니다: `@파일1 @파일2 비교해줘`

**꼭 기억하세요:** 한국어로 자연스럽게 말하면 됩니다!

> `0` = 목차 | 레슨 번호 입력 = 해당 레슨으로 이동 | **"완료"** = 이 레슨 완료 표시

"완료" 입력 시: `bash ~/.cc-start-kr/mark-complete.sh 3` 실행
