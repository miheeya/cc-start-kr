# 막혔을 때: 문제 해결 가이드

> 당황하지 마세요. 대부분의 문제는 간단하게 해결됩니다.

## 흔한 상황과 해결법

| 상황 | 해결 방법 |
|------|----------|
| Claude가 너무 오래 생각해요 | `Esc` 한 번 누르면 중단됩니다 |
| 잘못된 파일을 수정했어요 | `Esc Esc`(두 번 연타)로 되돌리기 (레슨 2) |
| 뭘 입력해야 할지 모르겠어요 | 한국어로 원하는 것을 설명하세요 |
| 권한을 물어보는데 뭔지 모르겠어요 | 잘 모르면 `n`(no)을 누르세요 — 안전합니다 |
| 화면이 너무 복잡해요 | `/clear`로 대화를 지우고 새로 시작하세요 |
| 이전에 한 작업을 다시 하고 싶어요 | `↑` 화살표로 이전 입력을 불러오세요 |
| 대화가 너무 길어졌어요 | `/compact`로 대화를 압축하세요 |
| 어제 하던 작업을 이어가고 싶어요 | `/resume`으로 이전 대화를 불러오세요 |

## 알아두면 좋은 추가 명령어

| 명령어 | 설명 | 언제 쓸까? |
|--------|------|-----------|
| `/permissions` | 권한 관리 | 매번 물어보는 게 귀찮을 때 |
| `/memory` | 규칙 편집 | Claude에게 규칙 가르칠 때 |
| `/init` | CLAUDE.md 자동 생성 | 새 프로젝트 시작할 때 |
| `/tasks` | 백그라운드 작업 확인 | 실행 중인 작업 목록 보기 |
| `/config` | 설정 변경 | 테마, 출력 스타일 바꾸기 |
| `/effort` | 사고 깊이 조절 | 간단한 질문엔 낮게, 복잡한 문제엔 높게 |
| `/fast` | 빠른 모드 전환 | 속도 우선일 때 |
| `/btw` | 사이드 질문 | 하던 작업 중 잠깐 다른 질문 |

## 에러 메시지가 나왔을 때

| 에러 메시지 | 의미 | 해결법 |
|------------|------|--------|
| Permission denied | 권한이 없음 | `y`를 눌러 허용하거나, `n`으로 건너뛰기 |
| File not found | 파일을 못 찾음 | `@`로 정확한 파일명 확인 |
| Context window full | 대화가 너무 길어짐 | `/compact`로 압축하거나 `/clear`로 새로 시작 |
| Rate limited | 요청이 너무 많음 | 1~2분 기다린 후 재시도 |
| Tool use denied | 도구 실행이 거부됨 | `Shift+Tab`으로 모드 확인 (plan 모드면 실행 안 됨) |

## 팁

- 가장 중요한 단축키 세 가지: `Esc`(중단), `Esc Esc`(되돌리기), `Shift+Tab`(모드 전환)
- `/help`를 입력하면 언제든 전체 도움말을 볼 수 있습니다
- 해결이 안 되면 `/doctor`로 설치 상태를 점검해 보세요
